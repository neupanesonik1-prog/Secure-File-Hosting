### Features Implemented (All 100% Working)
- User Registration & Login with JWT authentication
- Passwords securely hashed with bcrypt
- Protected routes (token stored in localStorage)
- Professional, responsive frontend (HTML + CSS + Vanilla JS)
- File upload page ready (PDF & MP4 only, max 20MB)
- Dashboard, My Files, Public Downloads pages
- Private files accessible only via shareable link
- File deletion (owner only)
- MongoDB Atlas / Railway hosted database (works from any computer)
- Clean folder structure & full commit history
